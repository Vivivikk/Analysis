# Data Analytics Portfolio
# Chocolate Sales & Marketing Performance

Репозиторий содержит комплексные решения для анализа бизнес-показателей: от глубокой сегментации базы данных на SQL до визуализации маркетинговых KPI на Python, Power BI.

## 🛠 SQL Analytics (PostgreSQL)
Технический фокус: Глубокая обработка данных, оконные функции и статистическое моделирование.

Ключевые возможности:
Anomaly Detection (Z-Score): Алгоритмический поиск рыночных аномалий через расчет стандартного отклонения волатильности.
Time Series Analysis: Реализация скользящих средних (SMA 50) и расчет накопительной доходности с момента IPO с помощью оконных функций (OVER, ROWS BETWEEN).
RFM-сегментация: Классификация рынков и клиентов на группы (Champions, At Risk) через статистическую функцию NTILE(3).
Data Transformation: Построение сложных аналитических витрин с использованием CTE для чистоты и масштабируемости кода.

Стек
PostgreSQL (Window Functions, CTE, Statistical Aggregations).

## Power BI: Дашборд по продажам
Цель: Визуализация ключевых показателей бизнеса для быстрого принятия решений.

Что сделано:
Главные метрики: Выручка, прибыль и количество заказов на одном экране.
Анализ товара: Графики по категориям (какой шоколад покупают лучше всего).
Продажи по странам: Интерактивная карта с географией заказов.
Динамика: Отслеживание роста или падения продаж по месяцам.
Фильтры: Быстрая сортировка данных по датам, менеджерам и регионам.

Стек
Power Query: Очистка и подготовка данных.
DAX: Расчет ключевых бизнес-показателей.
Data Modeling: Настройка связей между таблицами для правильной работы отчета.

## Marketing Dashboard (Python)
Цель: Автоматизация отчетности по рекламным кампаниям и расчет эффективности затрат.

Ключевой функционал:
Расчет KPI: Автоматический расчет Spend, Revenue, ROI и CPL (Cost Per Lead).
Cross-Platform Analysis: Сравнение прибыльности различных рекламных площадок.
Data Visualization: Построение графиков распределения прибыли и динамики затрат через Seaborn и Matplotlib.

Стек
  Python
  pandas
  matplotlib
  seaborn
